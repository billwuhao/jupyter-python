## bin() 整数的二进制形式

内置函数 bin()，Python 官方文档描述如下：


```python
help(bin)
```

    Help on built-in function bin in module builtins:
    
    bin(number, /)
        Return the binary representation of an integer.
        
        >>> bin(2796202)
        '0b1010101010101010101010'
    
    

返回给定整数的二进制表示形式的字符串。


```python
bin(123)
```




    '0b1111011'




```python
0b1111011
```




    123
