## str.upper 转大写

字符串方法 str.upper()，Python 官方文档描述如下：


```python
help(str.upper)
```

    Help on method_descriptor:
    
    upper(self, /)
        Return a copy of the string converted to uppercase.
    
    

返回原字符串的副本，其中所有区分大小写的字符均转换为大写。


```python
'嗨python'.upper()
```




    '嗨PYTHON'




```python
'πpython'.upper()
```




    'ΠPYTHON'
